# Дайджест рынка (RSS)

## Когда использовать
- Пользователь спрашивает о новостях рынка
- Нужен фундаментальный контекст
- Перед анализом рыночной ситуации

## Команда
```bash
npm run market:digest
```

## Что показывает
- Крипто-новости: BTC, ETH, DeFi, регуляция
- Forex-новости: экономические данные, решения ЦБ
- Макро-контекст: инфляция, ставки, геополитика

## Файлы
- `src/market/digest.ts` — сбор и парсинг RSS через `rss-parser`

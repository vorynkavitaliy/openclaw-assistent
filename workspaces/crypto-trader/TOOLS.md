# TOOLS.md — Crypto Trader Environment

## API Endpoints

### Binance
- REST API: https://api.binance.com/api/v3/
- WebSocket: wss://stream.binance.com:9443/ws/
- API Key: `[настроить в ~/.openclaw/openclaw.json]`
- API Secret: `[настроить в ~/.openclaw/openclaw.json]`

### CoinGecko (бесплатный, без ключа)
- API: https://api.coingecko.com/api/v3/
- Цены: /simple/price?ids=bitcoin,ethereum&vs_currencies=usd
- Маркеткап: /global

### TradingView
- Графики: https://www.tradingview.com/chart/
- Скринер: https://www.tradingview.com/crypto-screener/

## Инструменты

- **web_search** — поиск крипто-новостей
- **web_fetch** — загрузка данных с API
- **browser** — визуальный анализ графиков (TradingView)

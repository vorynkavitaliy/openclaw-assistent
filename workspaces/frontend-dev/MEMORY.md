## Уроки и инсайты

- Telegram Markdown: осторожно с символами `_`, `*`, `` ` `` — нужен escape
- Длинные сообщения (> 4096 символов) разбивать на части
- Для таблиц в Telegram использовать моноширинный шрифт (`` `код` ``)
- MarkdownV2 в Telegram требует escape многих символов — использовать HTML parse mode как альтернативу
